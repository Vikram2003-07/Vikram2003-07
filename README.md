### 🖥️ SIEM & Active Directory Home Lab

An enterprise-style **SOC Home Lab** built from scratch for hands-on attack simulation and detection engineering practice.

- 🏢 **Windows Server 2022** Domain Controller (ADDC01) with Active Directory at `192.168.10.7`
- 📈 **Splunk SIEM** for centralized log ingestion, SPL queries, custom dashboards, and alert creation
- 🐧 **Kali Linux** used to simulate attacker activity — brute force, lateral movement, credential attacks
- 🚨 Custom Splunk detections for authentication anomalies, failed logins, privilege escalation, and Windows event log patterns
- 📊 End-to-end SOC workflow: alert → triage → investigation → containment → documentation

🔗 [View Project](https://github.com/Vikram2003-07/Homelab)

### 🧠 KramCom CTF Platform

A custom-built **Capture The Flag platform** designed to simulate real-world cybersecurity challenges and train attacker mindset.

- 🚩 **8 progressive challenges** with realistic vulnerable environments
- ⚙️ Built end-to-end: challenge design, flag management, and deployment
- 🧪 Used internally at Vevion Technology Services as a security training resource

🔗 [View Project](https://github.com/Vikram2003-07/kramcom-ctf)

### 🚖 SecureRide – Vulnerable vs Secure Cab Booking Platform

A dual-version **web application** built to demonstrate OWASP Top 10 vulnerabilities and their mitigations side-by-side.

- ⚠️ **Vulnerable Version** — demonstrates XSS, SQL Injection, IDOR, and more
- 🛡️ **Secure Version** — same app rebuilt with security best practices applied
- 👥 Full-featured: authentication, ride booking, admin dashboard, simulated payment gateway
- 🎯 Built to understand how attackers exploit vulnerabilities and how defenders fix them

🔗 [View Project](https://github.com/Vikram2003-07/secureride)

---

### 🔐 BruteSafe – OSINT Security Toolkit

A browser-based **password intelligence & security toolkit** — fully client-side, no data leaves the browser.

- 🔐 Generates cryptographically strong passwords with entropy analysis
- 🕵️ Detects credential exposure in data breaches (privacy-safe, no plaintext sent)
- ⚡ Zero server-side processing — all analysis runs locally

🔗 [View Project](https://github.com/Vikram2003-07/BruteSafe)

Generate **strong, reproducible passwords** from a single master key using SHA-256. The same key always produces the same password — no database required.
